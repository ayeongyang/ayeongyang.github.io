[실습 03] Competent Cell 제작 및 대장균 형질전환(Transformation) Efficiency 분석

교과목명 : 분자생물학실습 [cite: 1]

실험 일자 : 2026년 5월 26일 / 2026년 6월 2일 [cite: 2]

보고서 작성자 : 학번 2624110086 / 양아영 [cite: 5]

1. 서론 (Introduction)

본 실험의 목적은 외부 DNA를 스스로 받아들이지 못하는 대장균을 화학적 처리를 통해 수용성 세포(Competent Cell)로 유도하고 [cite: 69], 플라스미드 벡터를 세포 내로 대량 주입하는 형질전환(Transformation) 공정을 마스터하는 것이다[cite: 68]. 실험을 통해 얻은 콜로니 수로부터 최종 형질전환 효율(Transformation Efficiency)을 정확히 산출한다[cite: 68, 116].

  • pET28a 벡터 : 카나마이신(Kanamycin) 저항성 유전자 (KanR)를 포함한다[cite: 8]. 선택 배지에 카나마이신을 첨가하여 해당 플라스미드를 가진 균주를 선택적으로 분리 배양한다[cite: 9].
  • pUC-GID 벡터 : 암피실린(Ampicillin) 저항성 유전자 (AmpR)를 포함한다. 선택 배지에 암피실린을 첨가하여 해당 플라스미드를 확보한 대장균만을 선별 유도한다 (★원문 오타 교정 완료).

2. 실험 재료 및 방법 (Materials & Methods)

[주요 재료] : LB 배지, 2X TSS Buffer (MgSO4, PEG-6000 포함), DMSO, 멸균수, 얼음, 42℃ 항온수조 [cite: 71, 72, 73, 80, 90, 104]

2.1 Competent Cell 준비 (Preparation)

  1. 단일 콜로니로부터 대장균을 37℃에서 O/N(Overnight) 진탕 배양하였다 (Preculture)[cite: 75].
  2. 5mL LB 배지에 전배양액 50µL를 접종한 후 본 배양을 수행했다[cite: 76].
  3. 배양액의 흡광도 OD600 값이 0.3 ~ 0.4 수준에 도달했을 때 성장을 급격히 중단시켰다[cite: 77].
    원리 : 세포 분열이 가장 활발한 대수생장기(Exponential phase)에는 세포막이 극도로 유동적이며 세포벽 구조가 완전히 완고해지지 않아, 화학적 충격(Heat shock)에 최적의 반응성을 나타낸다[cite: 78, 79].
  4. 배양액을 15mL 튜브로 옮겨 얼음 위에서 10분간 방치한 후 [cite: 80], 4℃ 상태에서 4,000rpm으로 5분간 원심분리하여 세포를 침전시켰다[cite: 81].
  5. 상층액을 날리고 멸균 처리된 차가운 LB 배지 625µL를 첨가해 세포 펠릿을 부드럽게 재부유시켰다[cite: 82].
    원리 : 후속 단계의 2X TSS 완충액을 최종 1X 농도로 희석하기 위함이다[cite: 83]. 극저온 상태를 유지해야 세포막의 무분별한 투과성 변형을 억제하고 균일한 컴셀 유도가 가능하다[cite: 83, 84].
  6. 차가운 상태의 2X TSS Buffer 500µL를 첨가하였다[cite: 85].
    원리 : TSS 내의 PEG(Polyethylene glycol)는 친수성 고분자로 대장균 주변의 수화층을 강제로 탈수시켜 DNA가 인지질 막 표면에 밀착하도록 유도하며 [cite: 86, 87], MgSO4의 2가 양이온은 DNA 뼈대의 음전하를 상쇄·중화하여 막 통과 능력을 향상시키고 외부 분해 효소를 억제한다[cite: 86, 88, 89].
  7. 실온 보관된 DMSO 125µL를 첨가하고 가볍게 혼합한 뒤, 차가운 튜브에 100µL씩 정밀 분주하였다[cite: 90, 93, 94].
    원리 : DMSO는 양친매성 유기용매로 세포막의 소수성 코어에 침투하여 지질 배열을 일시적으로 느슨하게 교란하고 미세 구멍을 형성한다[cite: 91]. 얼리면 응고되므로 실온 보관 상태의 액상을 사용해야 한다[cite: 92].

2.2 형질전환 공정 (Transformation)

  1. 제조된 컴셀 100µL 용액에 1~100µL 부피 내외의 플라스미드 DNA를 첨가하였다[cite: 94, 96, 97].
    원리 : 외부 DNA의 부피는 컴셀 용액의 1/10을 넘지 않아야 독성 및 염 농도 축적으로 인한 효율 저하를 막을 수 있다[cite: 97, 99].
  2. 얼음 위에서 30분간 조용히 방치하였다[cite: 101].
    원리 : 저온 조건 하에서 운동성을 잃은 DNA가 세포막 표면에 안정적으로 흡착 복합체를 형성하는 대기 단계이다[cite: 102, 103].
  3. 흔들림 없이 42℃ 항온수조에서 정확히 40초간 열 충격(Heat Shock)을 가했다[cite: 104].
    원리 : 42℃ 순간 고온에 의해 인지질 막이 Fluid 상태로 열리며 유도된 나노 기멍(Transient pore)을 통해 외부 플라스미드가 농도 기울기에 의거해 강제로 세포 안으로 유입된다[cite: 105].
  4. 얼음 위에 즉시 2~5분간 꽂아 급속 냉각을 수행하였다[cite: 107].
    원리 : 열린 막 구조를 즉각 닫아 진입한 플라스미드의 누출을 봉쇄하고 세포 손상을 리커버리한다[cite: 106, 108].
  5. 항생제가 없는 순수 LB 배지 500µL를 첨가하고 37℃에서 1시간 동안 흔들어 키웠다 (Recovery)[cite: 110, 112].
    원리 : 플라스미드 내의 항생제 저항성 단백질(AmpR, KanR)이 대장균 내부에서 실제로 발현 및 축적될 시간을 부여하는 과정이다[cite: 111, 113]. 이 단계 없이 바로 항생제 배지에 깔면 단백질이 없어 전멸한다[cite: 111].
  6. 항생제 고체 배지(LB Plate)에 300µL를 분주하여 고르게 도말한 뒤 배양하였다[cite: 114].

3. 결과 및 고찰 (Results & Discussion)

■ 수리적 효율 계산 수식 (Mathematical Formula)

Transformation Efficiency (TE) = 총 Transformant 수(CFU) / 사용한 DNA 양(µg) [cite: 117, 118]
1) 총 Transformant 수 = 콜로니 수 × (총 액체 부피 / 도말 부피) [cite: 121, 122, 123]
2) 사용한 DNA 질량 환산 공식 : ng 단위 ÷ 1,000 = µg 단위 (1 µg = 1,000 ng) [cite: 129, 135]
[그림 1] 항생제 배지 상에서 선별 성장한 pUC-GID 형질전환 대장균 콜로니 형상

■ 양아영 연구원 실제 분석 데이터 계산 과정 (TE Calculation)

- 확보된 최종 콜로니 수(Colony Count) = 219 CFU [cite: 142]
- 실험 총 액체 부피 = 100µL(컴셀) + 1µL(DNA) + 500µL(LB) = 601 µL [cite: 124]
- 플레이트에 도말한 부피(Spreading Vol.) = 300 µL [cite: 125, 142]

① 실제 총 Transformant 수(전체 부피 역산) :
  219 CFU × (601 µL / 300 µL) = 438.73 CFU [cite: 145]

② 사용한 플라스미드 DNA 질량 환산 (µg) :
  75.7 ng/µL 원액을 100배 희석하여 1µL 투입했으므로 실제 들어간 양은 0.757 ng이다[cite: 142, 143].
  0.757 ng ÷ 1,000 = 0.000757 µg

③ 최종 형질전환 효율 (Transformation Efficiency) 도출 :
  TE = 438.73 CFU / 0.000757 µg
  TE = 5.796 × 105 CFU/µg [cite: 147]
결과 해석 고찰 : 산출된 효율 $5.796 \times 10^5\text{ CFU/}\mu\text{g}$은 일반적인 화학적 수용성 세포 공정의 유효 신뢰구간에 정확히 안착하는 정상 수치이다[cite: 147]. 플레이트 상의 콜로니 분포가 뭉침 없이 균일하고 Smearing 오염이 발견되지 않은 점으로 미루어 볼 때, 2X TSS 완충액 배합 공정과 파이펫팅 희석 프로세스가 매우 숙련되고 고순도로 제어되었음을 증명한다[cite: 73, 143].
